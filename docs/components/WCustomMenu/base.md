# 自定义菜单

## 引入

```js
import { WCustomMenu } from '@yaoyaochi/weyui'
```

### 使用方法

:::demo
WCustomMenu/base
:::

### 配置参数

| 参数       | 说明   | 类型     | 可选值 | 默认值                        |
|----------|------|--------|-----|----------------------------|
| menuData | 菜单数据 | Object | —   | `{button:[]}` `根据微信官方结构使用` |


### 事件
| 事件名称       | 说明    | 回调参数                 |
|------------|-------|----------------------|
| submitData | 保存并发布 | <abbr>`event`</abbr> |
